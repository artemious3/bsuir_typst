# BSUIR typst template

Шаблон системы вёрстки Typst, почти соответсвующий [СТП 01-2024](https://www.bsuir.by/m/12_100229_1_185586.pdf). 

## Требования 

В системе должны быть установлены `Times New Roman` и `TeX Gyre Termes Math` (выбран в качестве наиболее 
близкого к Times New Roman шрифта с поддержкой математики) и непосредственно Typst.

## Сборка

В папке `examples` находятся примеры, которые собираются с помощью команды

```
typst compile examples/<...>/main.typ
```

## Замечания по использованию

- Есть автоматизированное оформление списка использованных источников с помощью файлов BibLaTeX или 
[Hayagriva](https://github.com/typst/hayagriva/blob/main/docs/file-format.md), но используется 
ГОСТ Р 7.0.5-2008 ([источник](http://mbio.bas-net.by/cager/ru/content/77-new-zotero-style)) 
В СТП 01-2024 требуется оформление по ГОСТ 7.1-2003.
Если требуется строгое соответствие этому ГОСТу, источники необходимо оформлять вручную.

- Реализованы многостраничные таблицы как требуется в пункте 2.6.4. Если нужно убрать 
нжнюю грань на не последних страницах таблицы, необходимо вручную найти место, где заканчивается таблица, 
и добавить там `table.hline(stroke:none)`, а также добавить в header `table.hline(stroke:black)`.
(см. [репродукцию Приложения Л](examples/stp_appendix/main.typ)).
